Quick start

1 · Login → 2 · Usa el access_token en cada request

# 1. Login POST http://localhost/SkyNet-SIA/public/api/v1/auth/login Content-Type: application/json { "name_user": "SuperAdmin", "password": "tu_password" } # Respuesta → tokens.access_token # 2. Usar el token GET http://localhost/SkyNet-SIA/public/api/v1/empleados Authorization: Bearer <access_token>
Niveles de acceso
público — sin autenticación api-key — header X-API-KEY jwt — cualquier usuario autenticado admin — nivel ≤ 2 superadmin — solo nivel 1
